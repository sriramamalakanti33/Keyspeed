# KeySpeed 🚀

A modern typing speed test application built with React and Vite.

## Features

- Real-time typing speed calculation
- WPM (Words Per Minute) tracking
- Accuracy calculation
- Timer-based typing tests
- Clean and responsive UI
- Instant performance statistics

## Technologies Used

- React.js
- Vite
- JavaScript
- CSS

## Installation

1. Clone the repository

```bash
git clone https://github.com/sriramamalakanti33/Keyspeed.git
```

2. Navigate to the project folder

```bash
cd Keyspeed
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
